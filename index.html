So how is this happening?

There are several things going on right now...

Model Conversion

Models are converted to formats that work in a JavaScript runtime:

  • ONNX
  • Tensorflow.js
  • Tensorflow Lite

There are hundreds of models out there ready to be used.

JavaScript Libraries

Libraries load and consume these models with JavaScript APIs:

  • Transformers.js
  • Tensorflow.js
  • MediaPipe

Runtime Options

Models can run on the CPU via WASM:

  • Browser
  • Web Worker
  • Server

GPU Acceleration with WebGPU

Models can run on GPU with new WebGPU APIs

Including platforms like Deno!

Experimental Chrome APIs

Chrome is experimenting with APIs to run system-shipped models

While JavaScript isn't the best language for AI, there are many cases where it makes sense and is the best tool for the job.

Tradeoffs

Tradeoffs

Should you be running this in the browser?

We talked about the positives - security, privacy, speed, etc.

But there are some tradeoffs to consider:

  • LLM's take up space
  • A paid LLM might be better
  • Using your user's CPU/GPU leaves much of the experience up to their device
  • Need to consider battery usage

Experimental Chrome APIs

Chrome is experimenting with APIs to run system-shipped models

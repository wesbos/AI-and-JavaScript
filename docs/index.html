Tradeoffs

There are some tradeoffs to consider:

  • LLM's take up space
  • Currently ONNX has a 2GB model limit
  • A paid LLM might be better
  • Using your user's CPU/GPU leaves much of the experience up to their device
  • Need to consider battery usage

While JavaScript isn't the best language for AI, there are many cases where it makes sense and is the best tool for the job.
